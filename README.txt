佳雨農場 單層安全版
====================

這一包沒有 index.html，不會再蓋掉首頁。

最簡單上傳方式：
1. 解壓 ZIP
2. 不要動你原本首頁
3. 把 vegetables.html 與所有蔬菜 .html 檔，上傳到網站同一層
4. 把 homepage-addon-only.html 裡的內容，貼進你原本首頁

正確結構：
/index.html        ← 你原本首頁
/vegetables.html
/butter-cabbage.html
/komatsuna.html
/baby-bok-choy.html
...其餘全部同層

這樣就不需要 recipes_full 資料夾。
